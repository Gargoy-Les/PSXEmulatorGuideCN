# 汉英术语对照表



transaction 事务&#x20;

byte array 字节数组

console 控制台

execution cycle 执行周期

masking 屏蔽

convention 约定

initializer 初始化程序

control logic \[自动化]控制逻辑

reset 复位

soft reset 软复位

memory map 内存映射

reset value 复位值

复位后，寄存器的值会恢复到初始且确定的复位值，以保证复位后每个功能模块的处于已知状态。

logic 逻辑；逻辑系统；逻辑运算

A system or set of principles underlying the arrangements of elements in a computer or electronic device so as to perform a specified task.

power cycling 电源重启

Assert&#x20;

Drive a signal to its active voltage level, either high or low. 将信号置于有效电平，不论高低

&#x20;De-assert

Drive a signal to its inactive voltage level, either high or low. 将信号置于无效电平，不论高低

程序计数器（Program Counter，PC）是一个中央处理器中的寄存器，用于指示计算机在其程序序列中的位置。在Intel x86和Itanium微处理器中，它叫做指令指针（instruction pointer，IP），有时又称为指令地址寄存器（instruction address register，IAR）

computer architecture 计算机体系结构

wrap

Cause (a word or unit of text) to be carried over to a new line automatically as the margin is reached, or to fit around embedded features such as pictures.

wrap back \[比喻义]

Cause (a word or unit of text) to be carried over to a new line automatically as the margin is reached, or to fit around embedded features such as pictures.
